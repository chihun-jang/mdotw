## 개인적으로 생각하는  md의 장점

+ 여러 플랫폼에서 md파일을 지원한다
	=> 블로그 이전과 같은 상황에 용이 
	
+  코드 편집기에서도 별다른 설정없이 md문법을 이용해 바로 글을 작성할 수 있다.
+ HTML로의 변환이 가능하다
+   수식 지원을 해줘 수식을 작성해야할 경우
